# ResGatinhos2.0
esGatinhos Blumenau é a plataforma que conecta gatinhos resgatados a lares em Blumenau/região. Focamos em adoção responsável, voluntariado e doações/apadrinhamentos, garantindo o bem-estar dos felinos. Acreditamos que todo gatinho merece uma 2ª chance. Simplificamos a adoção e apoiamos a causa animal.
